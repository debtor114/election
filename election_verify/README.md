# 6·3 지방선거 개표 검증 배터리

선관위 공개 개표 데이터로 떠도는 의혹을 **재현 가능한 통계 검정**으로 판정한다.
모든 검정은 (관측값)과 (우연 기대치/기준선)을 함께 출력하고, **PASS(정상) / FLAG(정밀조사)**
만 표시한다. 결론 문구는 하드코딩하지 않는다 — 숫자가 말한다.

> ⚠ **FLAG ≠ 부정의 증거.** FLAG는 "우연만으로 설명하기 어려우니 들여다볼 가치가 있다"는
> 뜻일 뿐이다. 다중비교(전국 수백 개 race를 동시에 보면 우연한 FLAG가 나옴)를 항상 감안할 것.

## 0. 데이터 받기 (T0)

```powershell
pip install requests pandas lxml numpy scipy matplotlib
# 전국·7종 전부 (시간 걸림)
python ..\nec_download.py --type all --out ..\nec_data
# 특정 선거만
python ..\nec_download.py --type 시도지사 --sido 전라남도
```

출력 구조:
```
nec_data/<선거종류>/<시도>_<구시군>[_선거구].csv   +   _combined.csv
nec_data/_ALL_combined.csv
```
tidy 스키마: `선거종류, 시도, 구시군, 선거구, 읍면동명, 구분, 선거인수, 투표수,
후보자별 득표수_<후보>..., 후보자별 득표수_계, 무효 투표수, 기권자수`

`구분` 값: `관내사전투표`, `선거일투표`, `계`(읍면동 합계) / 집계행은 `합계·거소투표·관외사전투표`.

선거종류: 시도지사·교육감·광역비례(시도단위) · 구시군의장·기초비례(구시군단위) ·
시도의원·구시군의원(선거구단위).

## 1. 검정 실행

```powershell
python run_all.py            # 전체 요약 리포트
python run_all.py 시도지사    # 특정 선거종류만

python tests\T1_duplicate.py 구시군의장 관내사전투표

```

| 검정 | 무엇을 보나 | FLAG 조건 | 비고 |
|---|---|---|---|
| **T1** 동일득표 | 읍면동 간 동일 득표 쌍(①한후보 ②상위2 ③복제) vs 부트스트랩 null | 상위2 z>+3, 방향 z>+3, 복제쌍≥1 | 출발점 `check_duplicate_votes.py` |
| **T2** 사전vs본 | 후보 사전·본 득표율 회귀의 완벽함 | R²>0.999 & 잔차<0.005 | 정상=잔차 흩어짐 |
| **T3** 투표율포렌식 | (투표율, 승자득표율) 산점도·2D히스토 (Shpilkin) | 꼬리상관>0.45 & lift>0.05 | `out/T3_*.png` 저장 |
| **T4** 끝자리/벤포드 | 득표수 끝자리 균일성·벤포드 1st | p<0.001 | **약함**, 단독 근거 금지 |

| **T6** 부족 vs 투표율 | 부족이 당일 고투표율로 설명되나 | 투표율 정상인데 부족한 읍면동 존재 | 이상치 `out/T6_*.csv` |

## 2. 컬럼/정당 매핑

- 후보 컬럼은 race마다 다르므로 `necdata.cand_cols()` 가 자동 식별(`후보자별 득표수_` 접두, `_계` 제외).
- T5의 정당 기준은 인자로: `python tests\T5_shortage_bias.py 구시군의장 국민의힘`
  (후보 컬럼명에 정당명이 포함됨: 예 `후보자별 득표수_더불어민주당 임지락`).

## 3. T5·T6용 부족 투표소 목록

`data/shortage_list.csv` (UTF-8) 직접 작성/수집:
```csv
시도,시군구,읍면동,투표소
인천광역시,연수구,송도2동,송도2동제3사전투표소
...
```
출처: 선관위 발표·언론·국정조사 자료. 읍면동 명칭은 개표 데이터와 일치시켜야 매칭됨.

## 4. 한계

- **다중비교**: 전국 수백 race를 보면 +3σ FLAG가 우연히 몇 건 나온다(look-elsewhere). FLAG 수 자체보다
  "기대 대비 초과분"과 재현성을 봐야 한다.
- **T4(끝자리/벤포드)**: 개표 득표수는 분포 특성상 벤포드를 잘 안 따른다 → 거의 항상 형식적 FLAG.
  보조 지표로만, 단독 근거 금지.
- **사전/본 분리**: `관내사전투표`·`선거일투표` 구분에 의존. `관외사전·거소`는 읍면동 미분해라 제외.
- **T5/T6**: 부족 투표소 목록의 품질·명칭 일치에 전적으로 의존.

## 5. 원칙
1. 관측값과 기준선을 항상 함께. "수상하다" 금지 → "관측 X, 기대 Y±Z, = Nσ".
2. 매핑(후보·정당·구분)은 설정/인자로 분리.
3. 결론 하드코딩 금지. 데이터가 말하게 한다.
