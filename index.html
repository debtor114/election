Bottom line

이건 "검증 도구"이지 "평결"이 아닙니다.

약 1,245개 선거단위 중 대부분 PASS(우연 모델로 설명되는 범위), 3개 FLAG(우연 모델의 기대치를 통계적으로 초과 → 더 볼 지점)입니다. FLAG도 PASS도 그 자체로 부정·결백의 증명이 아닙니다. 이 페이지는 "조작이 있다/없다"가 아니라 "어디가 기대치를 넘었는지"의 사실을 보여줍니다 — 해석은 읽는 사람의 몫이고, 데이터·코드·임계값을 전부 공개하니 직접 반박·재현하세요.

How it works

5가지 검정을 비유로

🎂

T1. 동일 득표 — 생일 역설

교실에 23명만 모여도 생일 같은 둘이 있을 확률이 50%. "똑같은 표가 있다"는 건 생각보다 자주 일어납니다. 실제 겹친 수를 우연이라면 나올 수와 비교합니다.

대부분 PASS 복제 사례 0건
📏

T2. 사전 vs 본투표 — 자로 그은 직선

모든 동네에서 "사전 = 본 + 일정값"처럼 자로 그은 듯 완벽하면 의심. 정상이면 들쭉날쭉 흩어집니다.

전부 PASS 완벽한 직선 0건
☄️

T3. 투표율×몰표 — 혜성 꼬리

투표함을 채우면 고투표율 동네일수록 특정 후보가 치솟는 '혜성 꼬리'가 생깁니다. 정상이면 둥근 구름.

7/7 둥근 구름
🎲

T4. 끝자리·벤포드 — 보조용

사람이 지어낸 숫자는 끝자리가 고르지 않은 경향. 단 개표수는 원래 잘 안 맞아 거의 항상 경고 → 단독 근거 금지.

약함
🗳️

T5·T6. 부족 투표소 — 목록 필요

"투표지 부족" 투표소가 특정 정당에 유리했는지, 단순 고투표율 탓인지 검정. 공식 부족 목록을 넣으면 작동.

About those flags

FLAG 3건 — 사실과 열린 해석

선거동일(상위2) 쌍기대치σ
전남 시도지사30.4+3.9
인천 시도지사10.02+7.0
충북 광역비례10.09+3.0
사실: 이 세 선거에서 동일 득표 쌍 수가 단순 우연 모델의 기대치를 통계적으로 유의하게 초과했습니다(그래서 PASS가 아니라 FLAG). 열린 해석: 이것이 다중비교에 따른 우연인지 (약 1,245개를 동시에 보면 +3σ가 몇 건 나오는 게 정상) 다른 원인인지는 이 검정만으로 단정할 수 없습니다. FLAG는 "여기를 더 보라"는 표시이지 결론이 아닙니다. 정상(PASS)을 "은폐"로, 이상치(FLAG)를 "스모킹건"으로 읽는 것 둘 다 데이터가 뒷받침하지 않는 비약입니다.

Forensic plots

투표율 × 승자 득표율 — 전부 '둥근 구름'

왼쪽 산점도/오른쪽 밀도. 부정선거 신호인 오른쪽 위 '혜성 꼬리'는 어디서도 없습니다.

Direct answers

떠돌던 사례들에 대한 답변

Q. "화순 이양면·강진 병영면, 민형배 둘 다 444!"

사실입니다. 데이터로 확인됩니다 — 민형배 후보는 2026 전남도지사 후보로, 전남 전역이 한 선거구라 두 곳 모두에 등장합니다. 관내사전투표 민형배 득표가 화순 이양면 444(투표수 578), 강진 병영면 444(투표수 654) 로 동일하며, 전남에서 민형배 관내사전이 444인 읍면동은 이 두 곳뿐입니다.

통계적으로: 이 전남 시도지사 race는 T1에서 FLAG됐습니다(상위2 동일 쌍 3개, 기대 0.4, +3.9σ). 단순 우연 모델의 기대치를 유의하게 초과한다는 뜻입니다. 다만 이것이 우연(다중비교)인지 다른 원인인지는 이 검정만으로 단정할 수 없습니다. 결과는 PASS도 조작 증명도 아닌 "추가 확인 필요" 상태입니다.

Q. "송도1·2동, 박찬대 3030 = 유정복 1440 동일!"

박찬대와 유정복은 서로 다른 선거(국회의원/광역단체장)의 후보로 알려져 있습니다. 서로 다른 선거의 두 숫자를 비교하는 경우와, 같은 선거 안에서 비교하는 경우는 통계적 의미가 다릅니다. 같은 선거 안에서 본 인천 시도지사 T1은 동일 쌍 1개로 FLAG됐습니다 (기대 0.02). 이 도구는 같은 선거 안에서의 동일 쌍을 기대치와 비교해 σ로 보고합니다.

위 수치는 모두 이 저장소의 데이터로 재현·반박할 수 있습니다. FLAG는 결론이 아니라 "들여다볼 지점"의 표시입니다.

Reproduce

직접 검증하기

1) 원본=가공본 무결성 확인 (숫자 안 건드렸나)

python verify_dataset.py

2) 공개 데이터로 검정 재현

pip install -r requirements.txt
cd election_verify && python run_all.py

3) 원본을 선관위에서 직접 받기

python nec_download.py --type all
다른 선거도: python nec_download.py --election-id <ID> --type <선거종류>
